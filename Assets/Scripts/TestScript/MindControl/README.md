# Test Scenario: Freeze Time Player Ability

This scene was created to test the freeze time ability of the player.

|Action | Keyboard |
|:-|:-:|
| Move | WASD |
| Reset Player | R |
| Mind Control Ability | 2 |
| Move Mind Control Selector | Mouse |
| Select Enemies | LMB |
